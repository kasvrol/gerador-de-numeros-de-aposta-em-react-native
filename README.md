<h1>Primeiros passos com o framework react native</h1>

<h3>Sobre</h3>
<p>Primeiro aplicativo feito com React Native usando como base a proposta de projeto do curso da Cod3r disponível no site da <a href="https://www.udemy.com/?utm_source=bing-brand&utm_medium=udemyads&utm_campaign=BG-Brand-Udemy_la.EN_cc.ROW&utm_term=_._ag_1219358808134939_._ad__._de_c_._dm__._pl__._ti_kwd-76210237821084_._li_147382_._pd__._&utm_term=_._pd__._kw_udemy%2B_._&matchtype=e&msclkid=e50a3ee4ec7a10a160555d588d0a9d28">Udemy</a>. O objeto era desenvolver um aplicativo que criasse um bilhete da mega-sena.</p>
<p>O foco do desenvolvimento do aplicativo foi sua funcionalidade principal utilizando a ferramenta e sem assistir as aulas de demonstração de uso desta, não sei o quanto o meu código se aproxima do código feito nas videoaulas, pois usei apenas a ideia de projeto. Fiz dessa maneira por 2 motivos: aprender a codar com utilizando a biblioteca apenas lendo a <a href="https://reactnative.dev/">documentação</a> e desenvolver a lógica do sistema.</p>
<p>O aplicativo foi criado com foco no android, pois é o sistema do dispositivo que utilizei durante o desenvolvimento.</p>
